# Employee Management System (ASP.NET Core MVC)

A production-style Employee & Department Management System built using ASP.NET Core MVC.

## 🔹 Features

- Department & Employee CRUD operations
- Entity Framework Core (InMemory DB)
- Search functionality
- Structured logging using Serilog
- Rolling file logs
- Session ID & Request Trace logging
- Admin dashboard UI with sidebar layout
- Dark mode toggle
- Process monitoring (Chrome processes)
- Clean and modular MVC architecture

## 🔹 Tech Stack

- ASP.NET Core MVC
- Entity Framework Core
- Serilog
- Bootstrap 5
- C#

## 🔹 Logging Capabilities

- Console logging
- Rolling file logging
- Session ID tracking
- Request Trace ID tracking
- Structured log format

## 🔹 Architecture

- Controllers (Business logic)
- Views (Razor UI)
- Models (Entities)
- DbContext (Data layer)
- Serilog (Logging layer)

---

This project demonstrates production-ready logging practices, structured application flow, and clean MVC architecture.
